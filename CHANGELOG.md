# 更新日志

本项目遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，版本号遵循[语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.1] - 2026-07-10

### 新增

- **编辑器 ↔ 预览滚动跟随**：编辑器顶栏新增「滚动跟随」开关（默认关闭），点击开启后，编辑与预览两栏按滚动*比例*双向同步（源码与渲染稿行高不同，按比例而非像素同步）。移动端为 Tab 切换、两栏不同屏，自动禁用。
- **查找 / 替换（Ctrl+F / Ctrl+R）**：用应用内 shadcn 组件自建编辑器顶部横条，替换 CodeMirror 自带的英文查找面板。
  - 中文文案、与应用设计系统一致、暗色模式下可读。
  - 实时 N/M 匹配计数（无结果显示「无结果」并红字提示，超 2000 条显示 `N/>2000`）。
  - 区分大小写 / 正则 / 全词三个开关。
  - 查找框 `Enter` 下一个、`Shift+Enter` 上一个；替换框 `Enter` 替换当前；`Esc` 关闭并焦点回编辑器。
  - 匹配高亮随查询实时刷新（自定义 ViewPlugin，因 CM 自带高亮仅在面板打开时生效）。

### 修复

- **链接全角冒号导致跳转异常**：中文输入法常把 `https://` 打成 `https：//`（全角冒号 `：`），浏览器识别不出协议，点击链接会跳到错误的相对地址。现在在 markdown-it 的 `normalizeLink` 里统一把全角 `：`／`／` 折回 ASCII，预览点击与复制到公众号的链接都恢复正常。

## [0.1.0] - 2026-07-05

- 首个版本：把 Markdown 转成内联 CSS 的 HTML，可直接粘贴进微信公众号编辑器。
- 三栏布局（编辑 / 预览 / 样式，桌面可拖拽分隔条 + 侧栏折叠，平板 2 列，手机 Tab）。
- 主题预设、样式调节面板、模板市场；导入 / 导出（Markdown / HTML / DOCX / Word）。
- Web 端与 Wails 桌面端同一份前端产物，运行时按 `window.go` 检测切换。
